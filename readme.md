# Proyecto de Sitio Web Personal - HTML y CSS

## 1. Descripción del trabajo

Este proyecto consiste en la creación de un sitio web personal que documenta y muestra los conocimientos adquiridos en la materia de HTML y CSS. El objetivo es demostrar competencia en el desarrollo de páginas web estáticas, estructura de contenido y estilos visuales mediante CSS.  

El sitio web incluye varias secciones que reflejan mi progreso académico y profesional, así como información personal y sobre el grado que estudio. La navegación es intuitiva y todas las páginas comparten un estilo común.

### Secciones del sitio web:

1. **Página principal (`index.html`)**: Introducción al sitio, enlaces a todas las secciones y presentación general.  
2. **Acerca de mí (`about.html`)**: Información personal, currículum vitae, imágenes y enlaces a redes sociales.  
3. **Tema de interés (`topic.html`)**: Información detallada sobre un tema elegido de interés personal o académico.  
4. **Red de compañeros (`net.html`)**: Lista de compañeros con enlaces a sus páginas web.  
5. **Grado (`degree.html`)**: Descripción detallada del grado, su estructura y lista de materias.  
6. **Fundamentos de la ingeniería informática (`fii.html`)**: Página dedicada a esta materia con objetivos y resultados de aprendizaje.  
7. **Contacto (`contacts.html`)**: Formulario básico para enviar mensajes (sin funcionalidad de envío).


---

## 2. Problemas durante el desarrollo

Durante la creación del sitio web, se encontraron los siguientes desafíos:

- **Enlaces relativos**: Ajustar rutas relativas para que las páginas dentro de `/public` y la página principal (`index.html`) carguen correctamente CSS e imágenes.  
- **Formulario de contacto**: Se diseñó un formulario funcional en apariencia, pero sin procesamiento de datos, cumpliendo con los requisitos del proyecto.  
- **Compatibilidad de estilos**: Lograr que todos los elementos visuales se vean consistentes entre las diferentes páginas, especialmente el menú de navegación y el footer.  

---

## 3. Tecnologías utilizadas

- **HTML**: Para la estructura de todas las páginas.  
- **CSS**: Para el diseño y estilo visual.  
- **GitHub**: Control de versiones y publicación del sitio web.

## 5. Conclusiones

El proyecto permitió:

- Aplicar y consolidar los conocimientos de HTML y CSS aprendidos en la materia.  
- Diseñar un sitio web coherente, organizado y visualmente atractivo.  
- Aprender a gestionar rutas relativas, estructura de carpetas y control de versiones con Git.  
- Desarrollar documentación clara y detallada sobre el proyecto y su desarrollo.  

El sitio web cumple con todos los requisitos del proyecto, incluyendo la organización de archivos, el uso de CSS externo, la navegación clara y la documentación en `README.md`.
